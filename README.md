# Rawai Renovation — GitHub Pages

## Publish
1) Create a new GitHub repo (example: `rawai-renovation`)
2) Upload these files:
   - index.html
   - styles.css
   - app.js
   - data.json
   - assets/ (optional)
3) In GitHub: Settings → Pages
4) Source: `Deploy from a branch`
5) Branch: `main` and folder `/root`
6) Save → your site URL will appear in Pages.

## Update content
Edit `data.json` to update:
- monthly timeline entries
- outstanding work items
- costs
- ownership
- rent and sale estimates
